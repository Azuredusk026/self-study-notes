# 第五章详细复习：常微分方程

## 1. 什么是微分方程？
方程里带导数（如 $y', \frac{dy}{dx}$）。你的任务是求出那个原函数 $y$ 是什么。

## 2. 一阶微分方程 - **必考解法 A & B**
### A. 可分离变量方程 (Separable Equations)
*   **长相：** $\frac{dy}{dx} = g(x)f(y)$ （$x$ 和 $y$ 能分家）。
*   **做法：** 把带 $y$ 的挪到一边，带 $x$ 的挪到另一边，两边同时积分。
    *   例：$\frac{dy}{dx} = xy \implies \frac{1}{y} dy = x dx \implies \ln y = \frac{1}{2}x^2 + C$。

### B. 一阶线性方程 (Linear Equations) - **套路化最强**
*   **标准型：** $y' + P(x)y = Q(x)$。
*   **做法（积分因子法）：**
    1. 计算积分因子 $I(x) = e^{\int P(x) dx}$。
    2. 通解公式：$y = \frac{1}{I(x)} \int [I(x) \cdot Q(x)] dx$。

## 3. 二阶常系数线性方程 - **特征方程法**
*   **长相：** $ay'' + by' + cy = 0$ ($a, b, c$ 都是常数)。
*   **解法：** 写出**特征方程** $ar^2 + br + c = 0$，解出两个根 $r_1, r_2$。
    1. **两个不同实根：** $y = C_1 e^{r_1 x} + C_2 e^{r_2 x}$。
    2. **两个相同实根 ($r_1=r_2=r$)：** $y = C_1 e^{rx} + C_2 x e^{rx}$ （别忘了多那个 $x$！）。
    3. **复数根 ($\alpha \pm i\beta$)：** $y = e^{\alpha x}(C_1 \cos \beta x + C_2 \sin \beta x)$。

## 4. 应用模型 (Modeling)
*   **指数增长：** $\frac{dP}{dt} = kP \implies P(t) = P_0 e^{kt}$。
*   **初值问题 (IVP)：** 题目会给你一个点（如 $y(0)=2$），目的是让你求出通解里那个常数 $C$ 的具体数值。

---
**本章解题锦囊：**
*   看到二阶方程 $y''$ 别慌，它比一阶方程更好解，因为不需要复杂的积分，只需要解二次方程（十字相乘或求根公式）。
*   解完方程，回头看看有没有给初始条件（如 $y(0)=\dots$），如果有，记得把 $C$ 算出来。
*   一阶线性方程第一步一定要先化成标准型（让 $y'$ 前面的系数为 1）。
