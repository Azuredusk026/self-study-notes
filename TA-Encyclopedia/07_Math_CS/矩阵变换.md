# 矩阵变换

## 定义与解释

矩阵变换是用矩阵表示平移、旋转、缩放、投影和坐标空间转换的数学方法。它是渲染管线和引擎 Transform 的基础。

## 核心原理

常见图形变换会把点或向量从 Object Space 变到 World、View、Clip 或 Screen Space。齐次坐标让平移可以写入 4x4 矩阵，矩阵乘法顺序决定变换组合顺序。

TA 需要理解点和方向的变换差异：点受平移影响，方向通常不受平移影响；法线在非均匀缩放下需要用逆转置矩阵处理。不同引擎的行列主序和乘法约定也必须确认。

## 用途

- 在图形、动画、碰撞、寻路、编辑器工具或调试可视化中解释与 矩阵变换 相关的结果。
- 把数学概念转成可验证的代码、Gizmos、Shader 计算或资源检查规则。
- 帮助 TA 与程序沟通空间关系、复杂度、数值稳定性和运行时限制。

## 与其他概念的区别

| 概念 | 区别 |
|---|---|
| [[四元数]] | 四元数更适合表示旋转，矩阵可表示完整空间变换。 |
| [[向量]] | 矩阵作用于向量或点来改变空间表示。 |

## 常见误区

1. 矩阵乘法顺序写反。
2. 把法线按普通位置点变换。
3. 忽略行主序/列主序和引擎约定差异。
