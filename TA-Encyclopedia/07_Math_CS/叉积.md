# 叉积

## 定义与解释

叉积是三维向量运算，结果是垂直于两个输入向量所在平面的向量，长度与两向量夹角的正弦和长度乘积相关。

## 核心原理

叉积方向由右手系或引擎坐标约定决定，常用于求法线、判断旋转方向、构建切线基底和计算面积。`a × b` 与 `b × a` 方向相反。

TA 使用叉积时必须确认坐标系手性和向量顺序。Unity、Unreal、DCC 和图形 API 在坐标约定上可能不同，顺序错了会导致法线反向、背面剔除或光照错误。

## 用途

- 在图形、动画、碰撞、寻路、编辑器工具或调试可视化中解释与 叉积 相关的结果。
- 把数学概念转成可验证的代码、Gizmos、Shader 计算或资源检查规则。
- 帮助 TA 与程序沟通空间关系、复杂度、数值稳定性和运行时限制。

## 常见误区

1. 忽略向量顺序导致方向相反。
2. 在二维问题中误用三维叉积语义。
3. 坐标系手性变化后没有重新验证法线方向。
