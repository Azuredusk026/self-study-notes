# 叉积

## 一句话定义

叉积用两个三维向量生成一个垂直于它们所在平面的新向量。

## 为什么需要它

叉积常用于计算法线、切线空间、旋转方向、三角形朝向和几何关系。TA 在处理模型法线、背面剔除、程序化网格和工具开发时会用到。

## 核心原理

- 输入：三维向量 A、B。
- 处理过程：按右手系或左手系规则计算垂直方向。
- 输出：与 A、B 都垂直的向量，长度等于平行四边形面积。
- 所在层级：数学、网格处理、Shader、DCC 工具。

```csharp
Vector3 normal = Vector3.Cross(edge1, edge2).normalized;
```

## 技术美术中的典型用途

- 由三角形边计算面法线。
- 构建 TBN 基底。
- 判断多边形绕序。
- 程序化生成网格。

## Unity 中的相关场景

Unity 使用 `Vector3.Cross`。注意 Unity 世界坐标系和图形 API 投影约定在不同上下文中的差异。

## Unreal Engine 中的相关场景

Unreal C++/Blueprint 提供 Cross Product，常用于方向和法线计算。

## 常见误区

1. 交换输入顺序会让结果方向相反。
2. 两个平行向量叉积接近零。
3. 忽略坐标系手性，导致法线反向。

## 面试可能怎么问

### 如何用三个点计算三角形法线？

回答要点：用两个边向量做叉积并归一化，注意顶点顺序决定法线方向。

## 实践建议

写一个程序化三角形网格，交换顶点顺序观察背面剔除和法线方向变化。
