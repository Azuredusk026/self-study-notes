# 点积

## 一句话定义

点积把两个向量映射成一个标量，用于衡量方向相似度或投影长度。

## 为什么需要它

光照强度、视线夹角、Fresnel、背面判断、投影、距离计算都常用点积。TA 写 Shader 时经常看到 `dot(N, L)`、`dot(N, V)`。

## 核心原理

- 输入：两个向量 A、B。
- 处理过程：对应分量相乘后相加；若都归一化，结果等于夹角余弦。
- 输出：标量。
- 所在层级：数学、Shader、游戏逻辑。

```csharp
float d = Vector3.Dot(a.normalized, b.normalized);
```

## 技术美术中的典型用途

- Lambert 漫反射。
- Rim Light 和 Fresnel 近似。
- 判断物体是否在角色前方。
- 计算投影和夹角。

## Unity 中的相关场景

Unity C# 用 `Vector3.Dot`，Shader 用 `dot(a, b)`。

## Unreal Engine 中的相关场景

Unreal Blueprint 有 Dot Product 节点，材质图也可使用 DotProduct 节点。

## 常见误区

1. 未归一化向量导致点积结果混入长度。
2. 不理解结果范围：单位向量点积在 -1 到 1。
3. 把点积当成距离。

## 面试可能怎么问

### dot(N, L) 在光照里表示什么？

回答要点：当 N 和 L 都归一化时，它表示法线和光方向夹角的余弦，可作为漫反射强度。

## 实践建议

做一个 Shader，把 `dot(N, V)` 映射成颜色，观察物体正面和边缘变化。
