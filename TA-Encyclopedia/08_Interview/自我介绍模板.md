# 自我介绍模板

## 30 秒版本

```text
你好，我主要方向是技术美术，关注 Shader、实时渲染、Unity/Unreal 工具和美术资产管线。
我做过 X 类型项目，主要负责 Y，例如 Z。
我希望在实习/岗位中继续做连接美术生产和引擎落地的工作。
```

## 90 秒版本

```text
你好，我的技术美术方向主要分三块：
第一是 Shader 和渲染效果，例如 X；
第二是工具和 Pipeline，例如 Y；
第三是资产规范和性能优化，例如 Z。

我最近做的一个项目是 A，当时遇到 B 问题，我用 C 方法解决，结果是 D。
所以我比较适合需要把美术效果、工具开发和引擎落地结合起来的 TA 工作。
```

## 准备要点

- 不要背履历，讲“方向 + 证据 + 目标”。
- 每个方向都准备一个项目例子。
- 不要夸大“负责整个管线”，讲清楚自己实际做过什么。

## 常见误区

1. 一上来讲太多课程和兴趣，缺少岗位相关性。
2. 只说会 Unity/Unreal，不说做过什么效果或工具。
3. 用“学习能力强”代替具体项目证据。
