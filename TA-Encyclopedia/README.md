## 维护原则

- 每个条目优先回答：是什么、核心原理、用途、常见坑、相关概念。
- 区分资料摘录、个人理解和工程验证结果。
- 新增术语时同步维护 `00_Index/术语索引.md`。
- 引用外部资料时登记到 `91_Sources/source_registry.md`。
